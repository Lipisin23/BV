Теория: Виет, остатки, прогрессии

x₁+x₂ = -b/a, x₁x₂ = c/a; x₁²+x₂² = S²-2P; x₁³+x₂³ = S³-3PS
Сумма коэфф. = P(1); сумма первых n нечётных = n²; телескоп: 1/(n(n+1)) = 1/n - 1/(n+1)

Задачи для самостоятельного решения (с полными ответами)

1. x² − 7x + 5 = 0, сумма корней?
Решение: по Виету x₁+x₂ = 7. Ответ: 7
2. x² − 4x + 3 = 0, произведение корней
Решение: x₁·x₂ = 3. Ответ: 3
3. Сумма коэффициентов (x²+x+1)⁵
Решение: P(1) = (1+1+1)⁵ = 3⁵ = 243. Ответ: 243
4. 1/x₁+1/x₂ для x²−5x+6=0
Решение: сумма=5, произв=6 ⇒ (5/6). Ответ: 5/6
5. Последняя цифра 23³
Решение: 3³=27 → последняя цифра 7. Ответ: 7
6. Сумма 1+3+5+...+999
Решение: n = (999+1)/2 = 500, сумма = 500² = 250000. Ответ: 250000
7. gcd(10¹²−1, 10⁸−1)
Решение: gcd(12,8)=4 → 10⁴−1 = 9999. Ответ: 9999
8. 100·(1/1·2 + ... + 1/99·100)
Решение: телескоп: (1 - 1/100) = 99/100, умножаем 100 ⇒ 99. Ответ: 99
9. Последняя цифра 1²+2²+...+2025²
Решение: формула n(n+1)(2n+1)/6, n=2025. mod10: 2025≡5,2026≡6,4051≡1 ⇒ (5·6·1)/6=5 ⇒ 5
10. При каком a сумма корней 2x²−ax+3=5?
Решение: сумма = a/2 = 5 ⇒ a=10. Ответ: 10
11. x₁²+x₂² для x²−4x+2=0
Решение: S=4,P=2 ⇒ 16-4=12. Ответ: 12
12. x₁³+x₂³ для x²−3x+1=0
Решение: S=3,P=1 ⇒ 27 - 3·3·1 = 27-9=18. Ответ: 18
13. Сумма коэфф. (x²⁰²⁵−x²⁰²⁴+1)²⁰²⁶
Решение: P(1)=(1-1+1)²⁰²⁶=1. Ответ: 1
14. (√x)⁴ + (√1-x)⁴ = 1
Решение: x² + (1-x)² = 1 → 2x²-2x+1=1 → 2x(x-1)=0 → x=0,1 сумма=1. Ответ: 1
15. Последняя цифра 1²+2²+...+1000²
Решение: 1000·1001·2001/6, mod10: 1000≡0 ⇒ число кратно 10 → последняя цифра 0. Ответ: 0
16. gcd(2¹⁵−1, 2⁹−1)
Решение: gcd(15,9)=3 → 2³−1=7. Ответ: 7

ML: регрессия, сигмоида, метрики

ŷ = w₀ + Σ wᵢxᵢ ; MSE = (1/n)Σ(y-ŷ)² ; Accuracy = TP+TN / all
σ(z) = 1/(1+e⁻ᶻ) ; класс = 1 если σ(z) > 0.5

📌 Задачи для самост. решения (с разбором)

ML1. ŷ = 1 + 2x₁ − x₂, x₁=3,x₂=4
Решение: ŷ = 1 + 2·3 − 4 = 1+6-4 = 3. Ответ: 3
ML2. ŷ = -2 + 5x, x=4
Решение: ŷ = -2+20=18. Ответ: 18
ML3. σ(0) = ?
Решение: σ(0)=1/(1+e⁰)=1/2=0.5. Ответ: 0.50
ML4. Логист. регрессия w₀=-1,w₁=2,x=1, порог 0.5
Решение: z = -1+2·1=1, σ(1)=0.731>0.5 ⇒ класс 1. Ответ: 1
ML5. MSE для y=[2,4,6], ŷ=[3,5,7]
Решение: ошибки: ( -1)²+(-1)²+(-1)² =3, MSE=3/3=1. Ответ: 1
ML6. Accuracy y_true=[1,0,1,1,0], y_pred=[1,1,1,0,0]
Решение: правильные: позиции 0,2,4 → 3 из 5 → 0.6. Ответ: 0.6
ML7. Подбор параметров: x=[1,2,3,4], y=[2,4,5,8], w₀∈[0,1], w₁∈[1,3] найти min MSE
Решение: перебор: (w₀,w₁)=(1,2) даёт ŷ=[3,5,7,9] MSE= (1+1+4+1)/4=1.75; (0,2) даёт [2,4,6,8] MSE=(0+0+1+0)/4=0.25 → лучший. Ответ: w₀=0, w₁=2, MSE=0.25
ML8. Визуализация ŷ=0.5+2x и точек (0.2,1),(0.4,2),(0.6,2),(0.8,3)
Решение (код): plt.scatter(x,y); plt.plot(x, 0.5+2*x); линия регрессии идёт почти через точки. График построен
ML9. R²: интерпретация
R² ∈ (-∞, 1]. Чем ближе к 1, тем лучше. Ответ: (−∞, 1]
ML10. Среднеквадратичная ошибка (формула)
MSE = 1/n Σ(y_i − ŷ_i)². Применяется для регрессии.
# Пример полного решения для ML7 (перебор параметров) import numpy as np x = np.array([1,2,3,4]); y = np.array([2,4,5,8]) best_mse = float('inf'); best_w = None for w0 in [0,1]: for w1 in [1,2,3]: y_pred = w0 + w1*x mse = ((y - y_pred)**2).mean() if mse < best_mse: best_mse, best_w = mse, (w0,w1) print(f"Лучшие w0={best_w[0]}, w1={best_w[1]}, MSE={best_mse}") # → (0,2), MSE=0.25

Код-шаблон для полного анализа данных (решения всех задач)

import pandas as pd user = pd.read_csv("user.csv") activity = pd.read_csv("activity.csv") message = pd.read_csv("message.csv") video = pd.read_csv("video.csv") # DA1-DA3 print("Всего пользователей:", len(user)) print("Указали дату рождения:", user['birth_date'].notna().sum()) print("Не указали пол:", user['gender'].isna().sum()) # Активности active_users = activity['user_id'].nunique() msg_cnt = (activity['activity_type'] == 'message').sum() print(f"Уникальных активных: {active_users}, сообщений: {msg_cnt}") # Сообщения >100 символов msg_merged = activity.merge(message, on='activity_id', how='inner') long_msgs = (msg_merged['num_characters'] > 100).sum() print("Длинных сообщений:", long_msgs) # Индекс активности def get_score(act): return {'post':3, 'comment':2, 'friendship':1}.get(act,0) activity['score'] = activity['action'].map(get_score) user_activity = activity.groupby('user_id')['score'].sum().reset_index() user_activity.rename(columns={'score':'activity_index'}, inplace=True) # средний возраст для индекса <5 user['birth_date'] = pd.to_datetime(user['birth_date']) user['age'] = (pd.Timestamp('2025-01-01') - user['birth_date']).dt.days // 365 df_full = user_activity.merge(user, on='user_id') low_activity = df_full[df_full['activity_index'] < 5] print("Средний возраст (активность<5):", low_activity['age'].mean())
